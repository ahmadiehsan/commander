# Commander

## Description

A framework for creating commands

## Usage

```
pip install git+<this/repo/url>.git@<version_tag>
commander-admin startproject <project_name>
```

## Available Commands

- `./run pre_commit.init`
- `./run pre_commit.run_for_all`
- `./run requirements.compile`
- `./run requirements.install.dev`
